TIPOS DE ESTILO



- ESTILO INLINE

O CSS inline é usado para dar estilo a um elemento HTML específico. Este tipo de CSS não é realmente recomendado, já que cada tag HTML precisa ser estilizada de maneira individual.

- ESTILO EXTERNO

Este tipo de uso de CSS em documentos HTML é o mais indicado e recomendado. Neste tipo, os códigos CSS ficam em um documento externo ao documento HTML, separando assim as camadas de estrutura e apresentação do projeto WEB.

- ESTILIZAÇÃO DE TEXTO

A tag "font" - altera o tipo de fonte de um texto em HTML.
Ex: font-family - permite que se faça uma lista de prioridades de familias de fontes e/ou nomes genéricos de famílias a serem especificados para um elemento selecionado.
font-size - é usada para indicar o tamanho da letra(fonte) em CSS.
font-weight especifica o peso ou a intensidade da fonte.

- BOX MODEL


Se refere a como os elementos HTML são modelados nos mecanismos do navegador e como as dimensões desses elementos HTML são derivadas das propriedades CSS.
