HTML


Sigla para HyperText Markup Language — Linguagem de Marcação de Hipertexto —. Com isso, o HTML é uma linguagem de marcação utilizada na construção de páginas na Web. O HTML é o componente base da webDocumentos e HTML pode ser interpretado por navegadores.

CONHEÇA OS ELEMENTOS BÁSICOS DO HTML

- head HTML

O elemento "head" usada para delimitar o cabeçalho do documento. Dentro dela outras tags de cabeçalho serão usadas, como o title, meta, link, entre outros.

- title HTML

O elemento HTML "title" (Elemento HTML Título) define o título do documento, mostrado na barra de título de um navegador ou na aba da página.

- body HTML

O elemento "body" do HTML representa o conteúdo de um documento HTML. É permitido apenas um "body" por documento.

- p HTML

O elemento HTML "p" representa um parágrafo.

- a HTML

O elemento HTML "a" (ou elemento âncora), com o atributo "href" cria-se uma hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL.

- br HTML

O elemento HTML "br" produz uma quebra de linha em um texto.

- img HTML

O elemento HTML "img" representa a inserção de imagem no documento, sendo implementado também pelo HTML5 para uma melhor experiência com o elemento "figure" e "figcaption".

- h1 - h2 - h3 - h4 - h5 - h6 HTML

Representam seis níveis de título de seção. O "h1" é o nível de seção mais alto e "h6" é o mais baixo.
